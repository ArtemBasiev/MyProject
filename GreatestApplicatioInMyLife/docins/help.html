Главное окно программы


Главное окно содержит следующие элементы:
Главное меню
Главное меню позволяет выполнять все основные операции.

Панель быстрого доступа
Набор клавиш, дублирующих некоторые пункты главного меню. Позволяет выполнять часто используемые операции быстрее, чем через главное меню.

Список документов
Список документов отображает документы для текущего склада за текущий рабочий период. Он имеет закладки, с помощью которых можно фильтровать список по видам документов.

Строка состояния
Строка состояния, при наличии соединения с базой данных, показывает следующую информацию: количество документов в списке, текущий рабочий период, его состояние, состояние клавиш CAPS LOCK и NUM LOCK

В названии главного окна указан текущий склад, то есть склад, который выбран для работы в данный момент
