Виды справочников


В программе используются следующие виды справочников:
  • Номенклатура

    • Справочник предназначен для занесения информации о номенклатуре, с которой работает предприятие.

  • Характеристика номенклатуры

    • Справочник предназначен для занесения информации о характеристике номенклатуры.

  • Контрагенты

    • Справочник предназначен для занесения информации о контрагентах предприятия.

  • Сотрудники

    • Справочник предназначен для занесения информации о сотрудниках предприятия, которые могут быть пользователями системы.

  • Объекты

    • Справочник предназначен для занесения информации об объектах, на которые осуществляется расход.

  • Единицы измерения

    • Справочник предназначен для занесения информации о единицах измерения номенклатуры.

  • Склады

    • Справочник предназначен для занесения информации о складах, имеющихся в организации.
