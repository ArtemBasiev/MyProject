Создание и редактирование документов



  • Создание документа

    • Для создания нового документа используйте кнопку "Добавить", расположенную в окне журнала. После чего в открывшемся окне заполните необходимые поля и нажмите кнопку "Создать"

  • Редактирование документа

    • Для правки данных документа кликните мышью на поле, которое необходимо редактировать, введите новые данные и нажмите клавишу "Enter".

  • Удаление документа

    • Для удаления документа выделите его в списке документов, кликнув на него мышью и нажмите кнопку "Удалить".

  • Добавление деталей документа

    • Для добавления номенклатуры в детали документа выделите нужный документ в списке, а затем нажмите на маленькую кнопку "Добавить", которая расположена ниже списка документов.

  • Удаление деталей документа

    • Для удаления деталей документа выделите ее в списке и нажмите кнопку "Удалить".

  • Редактирование деталей документа

    • Для правки детали документа выделите с помощью мыши поле, которое необходимо редактировать, введите новые данные и нажмите клавишу "Enter".Редактировать можно только количество номенклатуры .
