Главное окно программы


Главное окно содержит следующие элементы:
  • Главное меню

    • Главное меню позволяет выполнять все основные операции. Содержит следующие пункты:

    • Справочники

      • Предназначен для перехода к окнам справочников

    • Отчеты

      • Предназначен для перехода к окнам формирования отчетов

    • Рабочие периоды

      • Предназначен для перехода к окну управления рабочими периодами

    • Помощь

      • Предназначен для перехода к окну справки

    • Выход

      • Предназначен для выхода из приложения или из учетной записи



  • Панель документов

    • Содержит в себе вкладки журналов прихода, расхода и перемещения.
